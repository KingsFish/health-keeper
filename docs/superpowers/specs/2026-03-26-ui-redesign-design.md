# HealthKeeper UI 重设计方案

**日期**: 2026-03-26
**状态**: 已确认

## 概述

将 HealthKeeper 从当前的灰白朴素风格，重新设计为温暖安心、专业可靠的医疗档案管理界面。

## 设计目标

- **用户群体**: 家庭用户（管理自己和家人的病历）
- **设计风格**: 温暖安心、专业可靠
- **配色方向**: 柔蓝/天蓝系（平静、信任、医疗专业感）
- **字体选择**: Noto Sans SC（清晰易读，中文显示优秀）

## 布局方案

采用**仪表盘式布局**，进入即看到全家人健康状况概览，温暖直观。

## 核心页面设计

### 1. 首页（仪表盘）

- **顶部问候**: 渐变蓝背景，个性化问候，健康提示
- **快速操作**: 3 个大按钮，快速录入突出显示
- **家庭成员**: 横向卡片，选中成员蓝色高亮
- **时间线**: 最近就诊记录，时间轴可视化，成员筛选

### 2. 成员详情页

- **头部**: 渐变背景，头像、姓名、基本信息
- **健康卡片**: 慢性病、过敏、长期用药（彩色卡片区分）
- **就诊历史**: 列表形式，点击进入详情

### 3. 就诊记录详情

- **头部**: 医院、科室、医生、日期
- **信息卡片**: 诊断、主诉、治疗方案
- **药物列表**: 绿色背景卡片，显示用法用量
- **附件区域**: 缩略图预览

### 4. 快速录入流程

- **步骤 1**: 上传文件（拖拽或点击）
- **步骤 2**: AI 识别进度（OCR → 提取）
- **步骤 3**: 确认结果，预填表单

### 5. 搜索页面

- **搜索栏**: 输入框 + 取消按钮
- **筛选标签**: 成员筛选，横向滚动
- **结果列表**: 卡片式，关键词高亮
- **结果统计**: "找到 N 条相关记录"

### 6. 模态框样式

- **确认弹窗**: 图标 + 标题 + 描述 + 双按钮
- **表单弹窗**: 标题栏 + 表单区 + 底部按钮
- **成功提示**: Toast 风格，图标 + 文字
- **底部面板**: 上滑式，操作列表

## 设计细节

### 圆角

- 卡片: 12-16px
- 按钮: 8-12px
- 标签: 4-6px
- 头像: 50%

### 阴影

- 卡片: `0 1px 6px rgba(0,0,0,0.05)`
- 弹窗: `0 20px 40px rgba(0,0,0,0.15)`
- 按钮 hover: `0 2px 8px rgba(59,130,246,0.2)`

### 间距

- 页面内边距: 16-20px
- 卡片内边距: 12-16px
- 元素间距: 8-12px

| 状态 | 背景色 | 文字色 |
|------|--------|--------|

| 待处理 | #fef3c7 | #92400e |

## 字体规范
